# SlashBar

Barre de raccourcis pour Windows. Saisie d'une commande, puis exécution.

## Raccourcis

| Touche | Action |
|--------|--------|
| `Ctrl+Espace` | Ouvrir / fermer |
| `Ctrl+Shift+Q` | Quitter |
| `Esc` | Fermer |
| `Tab` | Compléter la commande ou le paramètre sélectionné |
| `↑` / `↓` | Parcourir les paramètres |
| `Entrée` | Exécuter |

## Modules

Voir [docs/](docs/README.md).

## Lancement

```powershell
cd SlashBar
.\run.ps1
```

Prérequis : .NET (Windows). Firefox doit être accessible dans le PATH pour le module `f`.
