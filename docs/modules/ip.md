# ip — Adresse IP

Copie une adresse IP dans le presse-papiers.

## Syntaxe

```
ip           → IP publique
ip local     → IP locale (LAN)
```

## Options

| Option | Description |
|--------|-------------|
| _(aucune)_ | Adresse IP publique (`api.ipify.org`) |
| `local` | Adresse IP locale (LAN) |

## Exemples

```
ip
ip local
```
